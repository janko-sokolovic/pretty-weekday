# Pretty weekdays v0.2.0

## Installation

In a browser:
```html
<script src="pretty-weekday.js"></script>
```

Using npm:
```shell
$ npm i -g npm
$ npm i --save pretty-weekday
```
In Node.js:
```js
// Load the full build.
var prettyWeekday = require('pretty-weekday');

prettyWeekday(1) // { name: "Monday", short: "Mon" }

```
