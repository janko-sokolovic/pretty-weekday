# Pretty weekdays v1.0.0

## Installation

In a browser:
```html
<script src="pretty-weekday.js"></script>
```

Using npm:
```shell
$ npm i --save pretty-weekday
```


## Usage
```js
var PrettyWeekday = require('pretty-weekday');

PrettyWeekday.toPrettyDay(1) // { name: "Monday", short: "Mon" }

```

## Test
```shell
$ npm test
```
